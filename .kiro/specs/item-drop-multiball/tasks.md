# Implementation Plan

 - [x] 1. 定数とデータ構造の追加
  - ドロップ率・上限・落下速度・アイテムサイズ・ボール上限・速度分岐定数を定義
  - Item構造体とゲーム状態へのスライス追加
  - _Requirements: 1,2,3,4_

 - [x] 2. ドロップ生成ロジックの組み込み
  - ブロック破壊イベントで確率判定し、上限未満ならアイテム生成
  - 生成座標を破壊ブロック中心に設定し初速を付与
  - _Requirements: 1_

 - [x] 3. アイテム落下・取得・消滅処理
  - 毎フレームの落下更新、AABBでパドル衝突判定し取得時に削除
  - 画面下到達で削除し、複数アイテムを独立更新
  - _Requirements: 2_

 - [x] 4. マルチボール適用処理
  - 取得時にボールを複製し2倍化、上限でクリップ
  - 新規ボールへ速度反転・分岐を割り当て、アイテムを消去
  - _Requirements: 3_

 - [x] 5. リセット／初期化整合
  - 新ゲーム/ラウンド開始と全ボール喪失時にボール配列とアイテムをリセット
  - 不変条件（1<=balls<=maxBalls, len(items)<=maxItems）を守る初期化
  - _Requirements: 4_

 - [x] 6. テスト追加
  - ドロップ判定上限、落下・取得・消滅、倍増と上限クリップ、リセットのテストを作成
  - _Requirements: 1,2,3,4_
