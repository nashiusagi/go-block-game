# Product Overview

Goで開発されたブロック崩しゲームです。プレイヤーはパドルを操作してボールを跳ね返し、画面上部に配置されたブロックを破壊するクラシックなアーケードゲームです。

## Core Capabilities

- **ブロック破壊**: ボールがブロックに当たるとブロックが破壊される
- **物理演算**: ボールが壁、ブロック、パドルに当たると光のように反射する
- **パドル操作**: 左右矢印キーでパドルを移動し、ボールを跳ね返す
- **ゲーム状態管理**: ゲームオーバー、勝利条件の判定

## Target Use Cases

- シンプルなゲーム体験を楽しむ
- クラシックなブロック崩しゲームの実装例として学習に使用

## Value Proposition

- 純粋なGo実装で、クロスプラットフォーム対応
- Ebitengineを使用した軽量で高速なレンダリング
- シンプルなコード構造で理解しやすい実装
